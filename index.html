Zik'Quest API

Enfin vous pouvez créer des blindtest en y voyant clair !

Simple et efficace


L'API du site Zik'Quest est une api permettant de crééer tout type de site de blindtest, liée avec l'API de spotify, elle est simple d'utilisation et plusieurs packs sont proposés selon l'utilisation voulue.

Basé sur l'API de Spotify


Le meilleur ? Zik'Quest API est basé sur l'API de Spotify, et ça vous garantit des musiques à l'infini ou presque.

Démonstration de L'API


Nous pouvons vous fournir une base pour la création de votre site de BLindTest et nous avons aussi la possibilité de de vous montrer un site fonctionel utilisant notre API. Ce dernier est inclus dans le Pack Maestro.
